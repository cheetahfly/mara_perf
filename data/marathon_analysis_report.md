# 马拉松成绩异常检测分析报告

**基于真实数据的统计学异常判定**

生成时间: 2026-05-22

---

## 研究问题

### 场景描述

一位30+岁男性业余跑者，
- 过去两年一直在为跑进 **2:20** 而努力（挣扎在 2:20 左右）
- 最近突然连续两次跑出惊人成绩：**2:17** 和 **2:13**

### 核心问题

> 这种情况在统计学上发生的可能性有多大？是否属于小概率事件？

---

## 数据筛选过程

**筛选流程**: 全马成绩 → 男性 → 30+岁 → 跑进2:30

| 筛选条件 | 说明 | 记录数 |
|---------|------|--------|
| 全马 | 排除半程马拉松 | - |
| 男性 | gender = M | - |
| 30+岁 | birth_year ≤ 1996 | - |
| 跑进2:30 | finish_seconds < 9000 | - |
| **符合条件的有成绩记录的运动员** | | **7,707 条** |

---

## 统计分析方法

### 1. 成绩换算为速度

将马拉松成绩换算为速度（km/h），便于直观理解：

```
速度 (km/h) = 42.195km ÷ (完赛时间秒数 ÷ 3600)
```

| 指标 | 数值 |
|------|------|
| 2:20 成绩对应速度 | **18.08 km/h** |
| 2:13 成绩对应速度 | **18.99 km/h** |
| 速度提升幅度 | **+5.0%** |

### 2. 改进幅度分布

计算所有运动员相邻两次比赛的成绩改进幅度（秒），负值=退步，正值=进步。

> 图表：改进幅度分布直方图
> - 红色虚线标记：用户场景（单次改进402秒）—— 位于分布最右侧极端位置

### 3. 概率计算

基于改进幅度分布，计算目标场景的概率：

```
用户场景: 2:20(8400秒) → 2:17(8177秒) → 2:13(7998秒)
总改进: 8400 - 7998 = 402秒
分两次: 第一次改进223秒, 第二次改进179秒
```

---

## 核心分析结果

### 基础统计

| 指标 | 数值 |
|------|------|
| 相邻成绩对样本量 | **5,662** |
| 平均改进幅度 | **-3.2秒** |
| 改进幅度标准差 | **133.7秒** |

### 单次改进概率

| 事件 | 改进幅度 | 观测次数 | 概率 |
|------|---------|---------|------|
| 单次改进 ≥ 402秒 | 402秒+ | 10 | **0.1766%** |
| 单次改进 ≥ 223秒 | 2:20→2:17 | 231 | **4.08%** |
| 单次改进 ≥ 179秒 | 2:17→2:13 | 413 | **7.29%** |

### 连续两次大改进概率

```
连续两次独立发生的概率 ≈ P(改进≥223秒) × P(改进≥179秒)
= 4.0798% × 7.2942%
= 0.297593%
```

### 实际观测结果

| 场景 | 实际观测 | 概率 |
|------|---------|------|
| 单次改进 ≥ 402秒 | 10 / 5,662 | **0.1766%** |
| 连续两次改进各≥180秒 | 1 / 5,662 | **0.0177%** |
| 两年内完成连续大进步 | 1 人 | **极端罕见** |

---

## 真实观测到的极端案例

在 5,662 对相邻成绩中，仅找到 **1例** 符合条件的连续大进步案例：

| 运动员 | 第一阶段 | 第二阶段 | 总改进 |
|--------|---------|---------|--------|
| **HAYLE LEMI** | 7912秒 → 7731秒 (改进 181秒) | 7731秒 → 7509秒 (改进 222秒) | **403秒** |

---

## 统计学结论

### 最终结论

> **用户场景** ("2:20 → 2:17 → 2:13 在两年内连续两次大进步")
> **在 5,662 对真实数据中仅观测到 1 例**
> **属于【极端小概率事件】**

### 异常判定

> 如果某运动员确实在两年内完成了从 **2:20** 到 **2:13** 的突破，
> 在统计学上属于【**异常值**】。
>
> 这种模式的概率 **< 0.0177%**，在真实数据中几乎从未发生过。

### 重要说明

**统计学异常 ≠ 证明作弊**

- 统计只能告诉我们"这很罕见"，不能证明"一定作弊"
- 可能存在其他合理解释：重大训练方法改变，体重显著下降、高原训练等
- 需要结合其他证据综合判断
